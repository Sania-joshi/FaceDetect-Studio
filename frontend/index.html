Workflow

How the app is structured

  1. Capture a webcam frame or upload an image.
  2. Send the image to a prediction service or browser model.
  3. Render the predicted emotion and confidence.
  4. Keep the UI responsive for Vercel deployment.
The current Python scripts stay useful for local training and validation, while this frontend becomes the presentation layer.
